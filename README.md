# imgcomp

A fast, private, client-side image compression app. Compress images directly in your browser — no uploads, no servers.

## Features

- Single and batch image compression
- Adjustable quality, output format, and EXIF stripping
- Drag-and-drop, file picker, and clipboard paste support
- Mobile-first, responsive design

## Tech Stack

Astro, TypeScript, Tailwind CSS, [browser-image-compression](https://github.com/nicolo-ribaudo/browser-image-compression)

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview
```
